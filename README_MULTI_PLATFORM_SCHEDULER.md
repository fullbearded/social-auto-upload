# 🚀 多平台智能视频调度发布器

## 📋 项目简介

这是一个功能强大的智能视频调度发布工具，支持8个主流社交媒体平台的自动化视频发布。采用先进的调度算法，实现智能化的内容分发管理。

## ✨ 核心功能

### 🌐 多平台支持（8个）
- **抖音** - 支持自定义封面，需要创作者账号
- **哔哩哔哩** - 需要biliup库，支持分区选择  
- **小红书** - 支持自定义封面和地理位置
- **快手** - 限制最多3个话题标签
- **百家号** - 支持AI成片功能，需要创作者权限
- **腾讯视频号** - 支持原创声明和商品添加
- **TikTok** - 使用Firefox浏览器，需要特殊网络环境
- **快手备用** - 功能类似主快手上传器

### ⏰ 智能调度系统
- **智能时间分配**：6:00、9:00、12:00、17:00、19:00
- **随机时间偏移**：0-10分钟随机延迟，避免固定模式
- **动态分组算法**：每5个视频为一组，优化发布节奏
- **余数智能处理**：自动处理非整除情况

### 🔀 随机化策略
- **视频顺序随机化**：避免固定发布顺序
- **时间随机偏移**：让每个视频的发布时间都不完全相同
- **智能时间槽分配**：动态分配最佳发布时间

### 📊 高级特性
- **多平台同时发布**：一次配置，多平台同步发布
- **交互式平台选择**：友好的用户界面
- **详细统计报告**：成功率、失败原因等
- **智能错误处理**：自动重试和错误恢复

## 🎯 使用场景

- **内容创作者**：管理多个平台的内容发布
- **MCN机构**：批量管理和发布视频内容
- **企业营销**：多平台品牌内容分发
- **自媒体运营**：提高内容发布效率

## 🚀 快速开始

### 1. 环境配置
```bash
# 安装依赖
pip install -r requirements.txt

# 安装浏览器环境
playwright install chromium firefox

# 配置环境文件
cp conf.example.py conf.py
# 编辑conf.py，设置Chrome路径
```

### 2. 内容准备
```bash
# 创建目录
mkdir -p videos cookies

# 放置视频文件到videos目录
# 可选：添加同名.png封面文件
# 可选：添加同名.txt标题标签文件
```

### 3. 生成Cookie
```bash
# 为需要的平台生成cookie
python examples/[平台]_uploader/[平台]_cookie.py
```

### 4. 运行发布器
```bash
python multi_platform_video_scheduler.py
```

## ⚙️ 配置选项

### 时间调度
- **发布时间**：可自定义5个时间点
- **随机偏移**：0-10分钟可调
- **开始延迟**：支持从未来某天开始

### 分组策略
- **分组大小**：每3-10个视频一组
- **余数处理**：自动分配到合适时间
- **优先级**：支持平台优先级设置

## 🎯 智能算法

### 时间分配算法
```
1. 获取用户设定的时间点
2. 为每个视频分配基准时间
3. 添加0-10分钟随机偏移
4. 确保时间不冲突
5. 生成最终发布时间列表
```

### 分组算法
```
1. 随机打乱视频顺序
2. 按设定大小分组
3. 处理剩余视频
4. 分配到合适时间槽
5. 生成完整发布计划
```
