# 多平台智能视频调度发布器使用说明

## 🎯 概述

`multi_platform_video_scheduler.py` 是一个功能强大的智能视频调度发布工具，支持 8 个主流社交媒体平台的自动化视频发布。该工具采用智能调度算法，支持随机打乱、动态分组、智能时间分配和随机时间偏移等高级功能。

## ✨ 主要特性

### 🌐 多平台支持

- **抖音 (douyin)** - 支持自定义封面，需要抖音创作者账号
- **哔哩哔哩 (bilibili)** - 需要 biliup 库，支持视频分区选择
- **小红书 (xiaohongshu)** - 支持自定义封面和地理位置
- **快手 (kuaishou)** - 限制最多 3 个话题标签
- **百家号 (baijiahao)** - 支持 AI 成片功能，需要百家号创作者权限
- **腾讯视频号 (tencent)** - 支持原创声明和商品添加
- **TikTok (tk)** - 使用 Firefox 浏览器
- **快手备用 (ks)** - 功能类似主快手上传器

### ⏰ 智能时间调度

- 默认发布时间：06:00、09:00、12:00、17:00、19:00
- 支持随机时间偏移：0-10 分钟随机延迟
- 智能避开固定时间，让发布更自然

### 📊 智能分组算法

- 每 5 个视频为一组，优化发布节奏
- 自动处理余数视频（41 个视频时，最后 1 个放在最后一天）
- 动态计算所需发布天数

### 🔀 随机化策略

- 随机打乱视频顺序，避免固定模式
- 随机时间偏移，避免被识别为机器人
- 智能分配发布时间槽位

## 🚀 快速开始

### 1. 环境准备

```bash
# 安装Python依赖
pip install -r requirements.txt

# 安装Playwright浏览器（必需）
playwright install chromium firefox

# 配置环境文件
cp conf.example.py conf.py
# 编辑conf.py，设置LOCAL_CHROME_PATH为你的Chrome路径
```

### 2. 准备视频文件

将你的视频文件（.mp4 格式）放入 `videos/` 目录：

```
videos/
├── video1.mp4
├── video2.mp4
├── video3.mp4
└── ...
```

### 3. 准备 Cookie 文件

为每个平台准备 cookie 文件，放在对应的目录：

```
cookies/
├── douyin_uploader/account.json
├── bilibili_uploader/account.json
├── xiaohongshu_uploader/account.json
├── kuaishou_uploader/account.json
├── baijiahao_uploader/account.json
├── tencent_uploader/account.json
├── tk_uploader/account.json
└── ks_uploader/account.json
```

Cookie 文件可以通过运行各平台的示例脚本来生成：

```bash
# 示例：生成抖音cookie
 python .\examples\get_tencent_cookie.py
```

### 4. 运行调度器

```bash
python multi_platform_video_scheduler.py
```

## ⚙️ 高级配置

### 自定义发布时间

编辑脚本中的 `daily_publish_times` 变量：

```python
daily_publish_times = [6, 9, 12, 17, 19]  # 修改为你想要的时间
```

### 调整分组大小

修改 `group_size` 变量来改变每组视频数量：

```python
group_size = 5  # 可以改为3、4、6等
```

### 设置延迟开始

修改 `start_days` 来设置从几天后开始发布：

```python
start_days = 0  # 0表示明天开始，1表示后天开始
```

### 调整随机时间范围

在生成调度计划时修改 `random_minutes` 参数：

```python
random_minutes=10  # 改为5、15、20等
```

## 🎬 视频文件要求

### 文件格式

- 主要支持：`.mp4` 格式
- 文件大小：建议不超过 500MB（具体限制因平台而异）
- 视频时长：各平台有不同限制

### 封面文件（可选）

- 文件名与视频相同，扩展名为 `.png`
- 例如：`video1.mp4` 对应 `video1.png`
- 尺寸要求：参考各平台推荐尺寸

### 标题和标签

- 使用与视频同名的 `.txt` 文件定义标题和标签
- 格式示例（video1.txt）：

```
这是视频标题
#标签1 #标签2 #标签3 #标签4
```

## 🔧 故障排除

### Cookie 验证失败

```
❌ douyin Cookie验证失败
```

**解决方法**：

1. 重新运行对应平台的 cookie 生成脚本
2. 检查 cookie 文件路径是否正确
3. 确认账号状态正常

### 上传失败

```
❌ 上传失败: [具体错误信息]
```

**常见原因**：

- 视频格式不支持
- 文件过大
- 网络连接问题
- 平台限制（如频率限制）
