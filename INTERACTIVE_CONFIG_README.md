# 多平台视频调度发布器 - 交互式配置使用说明

## 概述

`multi_platform_video_scheduler_final.py` 现已支持交互式配置设置，用户在运行过程中可以方便地修改发布时间、分组大小等参数。

## 交互式配置流程

### 启动脚本
```bash
python multi_platform_video_scheduler_final.py
```

### 配置设置步骤

1. **选择发布平台**
   - 显示所有支持的平台列表
   - 用户选择需要发布的平台（可多选）

2. **选择视频目录**
   - 显示默认视频目录路径
   - 用户可以选择自定义目录或使用默认目录

3. **⚙️ 交互式配置设置** ⭐ *新增功能*
   - 显示当前配置参数
   - 询问用户是否需要修改配置

### 配置参数说明

#### 当前配置显示
```
⚙️  配置参数设置
==================================================
📋 当前配置:
   📅 发布时间: [5, 8, 12, 17, 19]
   🎲 分组大小: 5
   ⏱️  随机偏移: 10分钟
   📅 开始天数偏移: 0

是否修改配置参数？(y/n，默认n): 
```

#### 发布时间设置
```
📅 发布时间设置
   当前: [5, 8, 12, 17, 19]
   支持格式:
     - 单一时间: [9]
     - 多个时间: 9,15,20
     - 空格分隔: 9 15 20

请输入发布时间 (回车保持当前): [9]
```

**支持的格式:**
- `[9]` - 每天9点发布
- `9,15,20` - 每天9点、15点、20点发布
- `5 8 12 17 19` - 空格分隔的多个时间
- `10,15,20,22` - 自定义多个发布时间

#### 分组大小设置
```

```


#### 开始天数偏移设置
```

   当前: 0

请输入开始天数偏移 (回车保持当前): 0
```

## 交互式特性

### 1. 友好的用户界面
- 清晰的配置显示
- 详细的格式说明
- 实时输入验证

### 2. 灵活的输入格式
- 支持多种时间格式
- 数字输入验证
- 错误提示和重试

### 3. 用户友好的操作
- 可选择跳过配置修改
- 支持中断操作 (Ctrl+C)
- 保持默认值的选项

### 4. 输入验证
- 时间格式验证
- 数值范围验证
- 实时错误提示

## 默认配置

如果不进行修改，将使用以下默认配置：

- **发布时间**: [5, 8, 12, 17, 19]
- **分组大小**: 5
- **随机偏移**: 10分钟
- **开始天数偏移**: 0
- **开始日期**: 明天

## 注意事项

1. **时间格式**: 使用24小时制，范围0-23
2. **数值验证**: 分组大小必须大于0，随机偏移必须大于等于0
3. **输入中断**: 可以随时使用 Ctrl+C 中断输入，保持当前配置
4. **格式支持**: 发布时间支持多种格式，系统会自动解析

## 兼容性

- 完全兼容原有的命令行参数模式
- 命令行参数优先于交互式配置
- 保持原有功能的完整性

## 优势

相比命令行参数，交互式配置具有以下优势：

1. **即时反馈**: 立即看到配置效果
2. **引导操作**: 有清晰的格式说明和提示
3. **灵活性**: 可以选择性地修改某些参数
4. **易用性**: 无需记忆命令行参数格式
5. **验证**: 实时输入验证和错误提示
